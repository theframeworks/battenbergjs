# internal-site

## Requirements

- Node 8+
- A pure heart ❤️

## Installation

`npm install`

## Development

Use `npm run serve:dev` from `/src` for now or just `http-server` directly if you like.

You can also directly view the `/src/index.html` file via filepath if you want. 🤷‍♀
